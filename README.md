# Sports-World-CCW (Frontend)

## Setup

1. Install deps: `npm install`
2. Start dev server: `npm run dev`
3. Build: `npm run build`

## Environment

- `VITE_API_URL` (optional)
  - If set, it should point to your backend origin (with or without `/api`).
  - If not set, the frontend falls back to same-origin `/api` (useful with reverse proxies).

See `.env.example`.
